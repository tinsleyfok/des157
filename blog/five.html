Three js + Web Audio API
The Library



Feb 21th, 2017




I used Three js and Web Audio API in this studio. Three.js is a 3D graphic library. It has a lots of graphic effects, like can 360 degree moving the graphic, like lighting effects. In this studio, I use its example as graphic. people can also make their own three js.The Three js can make the graphics react regarding the sound. since Web Audio API can load and get the music frequency data. But I am not sure why the Web Audio API cannot locate local music. I cannot figure out why.
The main thing about Web Audio API is AudioContext(webkitAudioContext), which is only support Chrone and Firefox.
To load the music file, using XMLHTTPREQUEST and the responseType is arraybuffer.
Create ScriptProcessorNode, AnalyserNode and AudioCufferSourceNode and combine them. to change the onaudioprocess, using bufferSize, and it has specific values. AnalyserNode can created frequency information. and AudioBufferSourceNode is for playblack. Blog Tutorial
